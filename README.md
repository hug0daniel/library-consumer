# 📚 library-consumer

A Kafka consumer application built with Spring Boot that listens to and processes messages from Kafka topics.

## 🚀 Features
- 🔄 Consumes messages from Kafka topics
- 📥 Processes incoming library-related events
- ⚙️ Built with Spring Boot for robust enterprise applications
- 🛠️ Easy to configure and extend

## 📋 Description
This repository contains a Kafka consumer service designed to consume and process messages as part of a library management system. 
It demonstrates how to integrate with Apache Kafka using Spring Kafka for reliable message consumption.

## 🔧 Technologies Used
- ☕ Java
- 🌱 Spring Boot
- 📨 Apache Kafka
- 🏗️ Gradle
